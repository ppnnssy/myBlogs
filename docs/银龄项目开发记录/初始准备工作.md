## 读代码
项目是一个 Vue3.3、TypeScript、Vite4、Pinia、Element-Plus 的后端管理项目，负责表格部分
表格使用的是Geeker-Admin，是一个对el-table二次封装的表格组件，封装程度非常高。
表格部分只需要写配置文件就够了。

## 遇到的问题
1. 代码格式化
项目使用eslint格式化代码，但是如果选择Vue - Official插件的格式化，就会和eslint冲突。所以要通过.settings.json文件选择默认格式化插件
其实还没有完全搞懂这个问题，不知道应该用什么插件格式化文档

2. 样式穿透。
使用element plus组件的时候需要对内部的样式做一些调整，用到样式穿透
Vue3和Vue2的样式穿透写法不一样：
```
<style scoped lang="scss">
.upload-demo{
  height:32px;
  display: flex;
  align-items: center;
  :deep(.el-upload-list) {
  display: flex;
  flex-wrap: nowrap;
  }
}
</style>
```
